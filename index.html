🖧 VLAN Configuration – Cisco Packet Tracer

This project demonstrates the configuration of two Virtual Local Area Networks (VLANs) using a Cisco 2960 switch and a Cisco 2811 router. The network is divided into two departments, HR and IT, each with its own IP subnet. Inter-VLAN routing was configured on the router to allow communication between the two VLANs.

  • Department: HR (VLAN 10)
    • PC0 – 192.168.1.10
    • PC1 – 192.168.1.20
  • Department: IT (VLAN 20)
    • PC2 – 192.168.2.10
    • PC3 – 192.168.2.20
  • 🖼️ Preview:
  • 📁 Download Project: Download .pkt
  • 🛠 Tools Used: Cisco Packet Tracer, Cisco 2960 Switch, Cisco 2811 Router
  • ✔ Skills Demonstrated:
    • Creating and naming VLANs
    • Assigning switch ports to VLANs
    • Static IP addressing
    • Inter-VLAN routing (Router-on-a-Stick)
    • Testing connectivity using ICMP (ping)
